What Breiman Learned as a Consultant: Good Guidelines

  1. Focus on finding a good solution—that's what consultants (also statisticians) get paid for.
  2. Live with the data before you plunge into modeling.
  3. Search for a model that gives a good solution, either algorithmic or data.
  4. Predictive accuracy on test sets is the criterion for how good the model is.
  5. Computers are an indispensable partner.

Quick Aside: Trees

Quick Aside: Random Forests

From http://www.stat.berkeley.edu/~breiman/RandomForests/cc_home.htm

Each tree is grown as follows:

  1. if the number of cases in the training set is N, sample N cases at random - but with replacement, from the original data. This sample will be the training set for growing the tree.
  2. If there are M input variables, a number m<<M is specified such that at each node, m variables are selected at random out of the M and the best split on these m is used to split the node. The value of m is held constant during the forest growing.
  3. Each tree is grown to the largest extent possible. There is no pruning.
